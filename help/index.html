Help

How do I upload a file?

Click "Choose Files" on the main page, select one or more files, then click "Upload". Once the upload is complete, you'll receive a download link.

Is there a file size limit?

No. There are no file size limits.

How long are files stored?

Files are kept as long as they are being accessed.

Can I add a note to my upload?

Yes. Click "Add Note" before uploading to attach a note that will be displayed on the download page.

How do I view my upload history?

Create an account or log in, then visit My Account to see all files you've uploaded.

What is the account ID?

Your account ID is a unique code that identifies your uploads. No password required — just keep it secret. You can create one for free on the Settings page.

How do I copy all download links at once?

After uploading multiple files, click the "Copy links" button to copy all download links to your clipboard at once.
