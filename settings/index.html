GitHub Settings

Files are uploaded as GitHub Release assets. You need a repository and a Personal Access Token (PAT) with repo scope (or public_repo for a public repo).

How to create a PAT

  1. Go to github.com/settings/tokens
  2. Click Generate new token (classic)
  3. Give it a name, e.g. Free Upload Manager
  4. Under Select scopes, check public_repo (public repos) or repo (private repos)
  5. Click Generate token and paste it above

Your token is stored only in your browser's localStorage and is never sent anywhere except GitHub's API. Files in a public repository are publicly accessible by anyone with the link.
