Free Upload Manager stores your files as GitHub Release assets — no external service required.

How it works

  • Files are uploaded as GitHub Release assets — stored directly in your repository.
  • Download links are permanent GitHub URLs, independent of any third-party service.
  • No file size limits beyond GitHub's 2 GB per asset.
  • All uploads require your own GitHub token — your files, your control.
  • Upload history is saved locally in your browser.

Files uploaded to public repositories are publicly accessible. Use a private repository if you need restricted access.
